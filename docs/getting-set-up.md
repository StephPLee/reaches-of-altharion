# Character Creation

# Character Creation Guide

This is a step by step guide on getting your character created and playable on the server. 

## Getting set up

First things first, you need to log into [WestMarches.games](https://www.westmarches.games/communities/reaches-of-altharion) using your discord account. This is the website we use to organise a lot of the mechanical stuff that goes into running a west marches game. You will need this account to use your character in game sessions and make in character purchases. 

We allow character sheets from DnD Beyond, Dicecloud, and GSheets. If you are using DnD Beyond, we have several campaigns for content sharing, access to the allowed partnered content and server homebrew. We have a Discord bot that distributes the links which can be used inside the server using /cc-link. 

Additional creation rules: 

Your character must comply with the 2024 PHB rules and use 5.5e content. You may use the 2014 version of content that doesn’t have a 2024 reprint.

You can choose to start your character anywhere between level 1 and level 4\. If you choose to start above level 1, when creating your character on WestMarches.games add the initial starting xp on creation

| 2nd level | 3rd level | 4th level |
| :---- | :---- | :---- |
| 300xp  | 900xp | 2700xp |

You may only multiclass your character twice which means a total of 3 classes across 20 levels

You start with one character slot, but you gain more as your characters reach certain milestone levels according to the table

| 5th level | 9th level | 13th level | 17th level |
| :---- | :---- | :---- | :---- |
| 2 slots | 3 slots | 4 slots | 5 slots |

## Creating your character

### Your character’s stats

You can generate your characters stats using the following methods and at any point while creating your character you may change which stat generation method you use. 

Server standard array \- 17 16 14 13 12 8  
Point buy with 40 points \- [This link](https://chicken-dinner.com/5e/5e-point-buy.html#select&NA&8&8&8&8&8&8&0&0&40&17&8&19&15&12&9&7&5&4&3&2&1&0&1&2&4&6&9&4&4&4&4&4&4) is already set up with a 40 point calculator  
Rolling for stats \- Roll in [Character Rolls](https://discord.com/channels/1417076658460033077/1417136305535451137)  
The first time you roll you can roll 5 times. However if after rolling or using the above methods you don’t have a stat line you like, you may wait 48 hours and roll 4 more times. This can be repeated every 48 hours. 

Just copy the full multiline command and Avrae will roll the appropriate number of times. 

| First Roll | After 48 hours |
| :---- | :---- |
| \!multiline  \!rollstats  \!rollstats  \!rollstats  \!rollstats  \!rollstats | \!multiline  \!rollstats  \!rollstats  \!rollstats  \!rollstats  |

### Background

When choosing your background, we use Tasha’s rules so you may choose any allowed feat not strictly the predetermined origin feat. 

All backgrounds give your choice of \+2/+1 or \+1/+1/+1 to any stats

### Starting Grace

On top of your starting feat, every new character gains a Starting Grace. These graces are listed [here](https://www.westmarches.games/communities/reaches-of-altharion/world/starting-graces). They can be added on DnD Beyond as feats. 

### Starting Equipment vs Gold

You can choose either equipment or gold when creating your character, but if you choose gold please do not purchase items until after your character is approved. You can then do so using the in site marketplace using the gold you gained at creation. 

You can add your starting gold when submitting in the same way as your starting xp. 

## Submitting Your Character

After finishing your character you can submit it on WestMarches.games and it will automatically submit it to the discord server in [Characters](https://discord.com/channels/1417076658460033077/1417089324041764934) to be reviewed. You can do this from the Characters tab. When filling out starting xp and gold it also shows the stellar coins option, you start with 0 stellar coins.  

Please include the following information in your Description or Player Notes to GM section when submitting 

Age:   
Starting level:  
Backstory:  
Background:   
"Origin" Feat:  
Starting Grace:  
How you rolled: \[ rolled in character-rolls / used standard+ array / used point buy \]  
       \*Leave one option

If you rolled please include a link to the message in Character Rolls with your stat roll of choice. 

A member of staff or DM on the server will then assess your submission, check the character is correct and let you know if you have missed anything.
