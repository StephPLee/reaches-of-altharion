# Getting set up

## Reaches of Altharion Character Creation Guide

This is a step-by-step guide on getting your character created and playable on the server.

First things first, you need to log into WestMarches.games using your Discord account. This is the website we use to organize a lot of the mechanical stuff that goes into running a west marches game. You will need this account to use your character in game sessions and make in-character purchases.

We allow character sheets from DnD Beyond, Dicecloud, and GSheets. If you are using DnD Beyond, some campaigns are set up to join for content sharing, so making your character in one of those will give you access to all the allowed partnered content and server homebrew. ### Additional creation rules

- Your character must comply with the 2024 PHB rules and use 5.5e content. You may use the 2014 version of content that does not have a 2024 reprint.
- You can choose to start your character anywhere between level 1 and level 4. If you choose to start above level 1, when creating your character on WestMarches.games add the initial starting XP on creation.
- You may only multiclass your character twice, which means a total of 3 classes across 20 levels.
- You start with one character slot, but you gain more as your characters reach certain milestone levels according to the table.

# Creating your character

## Your character's stats

You can generate your character's stats using the following methods, and at any point while creating your character you may change which stat generation method you use.

- Server standard array: 17, 16, 14, 13, 10, 8
- Point buy with 40 points: this link is already set up with a 40-point calculator
- Rolling for stats: roll in Character Rolls

The first time you roll, you can roll 5 times. However, if after rolling or using the above methods you do not have a stat line you like, you may wait 48 hours and roll 4 more times. This can be repeated every 48 hours.

Just copy the full multiline command and Avrae will roll the appropriate number of times.

## Background

When choosing your background, we use Tasha's rules, so you may choose any allowed feat, not strictly the predetermined origin feat.

All backgrounds give your choice of +2/+1 or +1/+1/+1 to any stats.

## Starting Grace

On top of your starting feat, every new character gains a Starting Grace. These graces are listed here. They can be added on DnD Beyond as feats.

## Starting Equipment vs Gold

You can choose either equipment or gold when creating your character, but if you choose gold please do not purchase items until after your character is approved. You can then do so using the in-site marketplace using the gold you gained at creation.

You can add your starting gold when submitting in the same way as your starting XP.

# Submitting Your Character

After finishing your character, you can submit it on WestMarches.games and it will automatically submit it to the Discord server in Characters to be reviewed. You can do this from the Characters tab. When filling out starting XP and gold it also shows the Stellar Coins option; you start with 0 Stellar Coins.

Please include the following information in your Description or Player Notes to GM section when submitting:

- Age
- Starting level
- Backstory
- Background
- "Origin" Feat
- Starting Grace
- How you rolled

Use one of these roll method notes:

- "rolled in character-rolls"
- "used standard+ array"
- "used point buy"

If you rolled, please include a link to the message in Character Rolls with your stat roll of choice.

A member of staff on the server will then assess your submission, check the character is correct, and let you know if you have missed anything.
